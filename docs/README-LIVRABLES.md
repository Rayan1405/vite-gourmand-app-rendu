# Dossier Livrables ECF

Ce dossier contient les documents demandes pour le rendu ECF.

## Contenu

- `charte-graphique.md` : palette de couleurs, typographies, regles UI
- `maquettes.md` : description des maquettes desktop et mobile
- `maquettes.html` : planche visuelle wireframes + mockups
- `maquettes.pdf` : export PDF des maquettes
- `gestion-projet.md` : organisation, backlog, suivi, risques, tests
- `documentation-technique.md` : choix techniques, environnement, modelisation, deploiement
- `manuel-utilisateur.md` : presentation de l application + parcours + comptes de test
- `livrables-liens.md` : liens a renseigner (GitHub, deploiement, gestion de projet)
- `diagrammes/cas-usage.mmd` : source Mermaid du diagramme de cas d usage
- `diagrammes/mcd.mmd` : source Mermaid du MCD
- `diagrammes/sequence-commande.mmd` : source Mermaid du diagramme de sequence
- `diagrammes/diagrammes.html` : document unique regroupant les 3 diagrammes
- `diagrammes/diagrammes.pdf` : export PDF unique des 3 diagrammes
